# Project Core Mandates

## 1. Do No Harm to Working Features
- **Strict Rule:** Whenever making edits, DO NOT make changes to features that are already working well.
- **Scope of Modification:** Keep changes surgical and strictly confined to the specific file, function, or line that requires the fix or feature addition.
- **Refactoring:** Avoid opportunistic refactoring or "cleanups" of surrounding code unless explicitly requested by the user. If a piece of code works, leave it alone.
